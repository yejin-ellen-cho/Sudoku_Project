# 🧩 스도쿠 게임

스도쿠는 가로, 세로, 그리고 3×3의 작은 정사각형 안에 1부터 9까지의 숫자를 **중복되지 않게 채워 넣는 논리 퍼즐 게임**입니다.  
플레이어는 주어진 힌트를 기반으로 규칙을 만족하도록 빈 칸을 채워야 합니다.

---

## 📌 규칙

- 각 **행(row)**에는 1부터 9까지의 숫자가 중복 없이 들어가야 합니다.
- 각 **열(column)**에도 1부터 9까지의 숫자가 중복 없이 들어가야 합니다.
- 각 **3×3 정사각형 박스** 안에도 1부터 9까지의 숫자가 중복 없이 들어가야 합니다.

---

## 🛠️ 주요 기능

### 1. `initialize_board`
- 빈 스도쿠 보드를 생성합니다.
- 9×9 형태의 초기 상태를 구성합니다.

### 2. `print_board`
- 현재 스도쿠 보드를 출력합니다.
- 사용자에게 현재 상태를 시각적으로 보여줍니다.

### 3. `solve_sudoku`
- 주어진 스도쿠 퍼즐을 자동으로 해결합니다.
- 백트래킹 등의 알고리즘을 활용할 수 있습니다.

### 4. `generate_sudoku`
- 완성된 스도쿠 보드에서 일부 값을 제거하여 퍼즐을 생성합니다.
- 난이도 조절이 가능하도록 확장할 수 있습니다.

### 5. `play_sudoku`
- 사용자가 직접 스도쿠를 플레이할 수 있는 인터페이스를 제공합니다.
- 게임은 다음 경우 종료됩니다:
  - 사용자가 잘못된 값을 입력한 경우
  - 퍼즐을 완성한 경우
